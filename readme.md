# Servicio cliente-servidor GRPC

Ejemplo sencillo de GRPC

## Tech Stack

**Server:** Node, GRPC

## Ejecutar localmente

Clonar proyecto

```bash
  git clone https://github.com/kioshiokamoto/grpc-ejemplo-sencillo
```

Ir a ruta de proyecto

```bash
  cd grpc-ejemplo-sencillo
```

Instalar dependencias

```bash
  npm install
```

Iniciar servidor

```bash
  node server
```

Iniciar cliente

```bash
  node client 1
```
